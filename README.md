📊 Retail Sales Dashboard & Forecasting (Power BI)

📌 Project Overview

This project presents an interactive Retail Sales Dashboard built using Power BI to analyze sales performance, profitability, customer segments, and shipping trends.

The dashboard also includes a 15-day sales forecasting model that predicts future sales based on historical order data.

The goal of this project is to demonstrate data visualization, business intelligence, and forecasting techniques using Power BI.

🚀 Features

1️⃣ Sales Performance Overview

Total Sales

Total Profit

Total Quantity Sold

Average Delivery Time

These KPIs provide a quick snapshot of overall business performance.

2️⃣ Regional Filtering

Users can filter the dashboard by region:

Central

East

South

West

This allows analysis of regional sales trends and performance.

3️⃣ Geographic Sales Analysis

A map visualization shows:

Sales distribution across U.S. states

Profit distribution geographically

This helps identify high-performing states and regional opportunities.

4️⃣ Category & Sub-Category Analysis

The dashboard analyzes product performance through:

Sales by Category

Office Supplies

Furniture

Technology

Sales by Sub-Category

Chairs

Binders

Phones

etc.

5️⃣ Customer Segment Analysis

Sales are broken down by customer segment:

Consumer

Corporate

6️⃣ Payment Mode Analysis

COD

Online

Cards

7️⃣ Shipping Mode Analysis

Sales distribution based on shipping method:

Standard Class

Second Class

First Class

Same Day

This helps evaluate logistics usage patterns.

8️⃣ Monthly Sales & Profit Trends

Time-series visualizations track:

Monthly sales growth

Monthly profit trends

This allows businesses to identify seasonal trends and growth patterns.

🔮 Sales Forecasting

The dashboard includes a 15-day sales forecast using Power BI's built-in forecasting model.

Forecast visualization includes:

Predicted sales values

Confidence interval range

Historical trend comparison

This helps estimate future sales performance and demand patterns.
